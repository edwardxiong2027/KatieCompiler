GitHub sync

Save and load your programs straight from a GitHub repository. This runs in your browser using a personal access token — nothing is sent anywhere except GitHub.

How do I get a token?

On GitHub: Settings → Developer settings → Fine-grained tokens → Generate new token. Limit it to the one repository you want to use, and give it Contents: Read and write. Paste it above. It is stored only in this browser's local storage.

Settings

Runs entirely in your browser with Pyodide (CPython on WebAssembly). Most Python works, including NumPy, pandas and matplotlib. What can't work in a browser: desktop GUI windows (tkinter, pygame), real network sockets, subprocess, and multiprocessing.
